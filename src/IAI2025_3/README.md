## 问题描述

利用决策树判断是否需要在餐馆等座位。基于以下属性进行决策：

1. **Alternate（改变）**：附近是否有另一家合适的餐馆
2. **Bar（酒吧）**：该餐馆中供顾客等候的酒吧区是否舒适
3. **FriSat（周五周六）**：若是周五或周六，则为真
4. **Hungry（饥饿）**：我们是否饥饿
5. **Patrons（顾客）**：该餐馆中有多少顾客（None-没人，Some-一些，Full-满座）
6. **Price（价格）**：餐馆的价格范围（$，$$，$$$）
7. **Raining（下雨）**：外面是否在下雨
8. **Reservation（预约）**：我们是否预约过
9. **Type（类型）**：餐馆的种类（法式、意大利式、泰式或汉堡店）
10. **WaitEstimate（等候时间估计）**：餐馆主人估计的等候时间（0-10分钟，10-30，30-60，>60）
